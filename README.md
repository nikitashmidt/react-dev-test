# Тестовое задание Frontend разработчик на React.

Вам дан проект Дашборда со списком звонков, для операторов, в котором они могут просматривать детали звонка и менять их статусы.

```
Проект уже работает, но в нем намеренно допущены ошибки, которые вам предстоит выявить и исправить.
```

## Что нужно сделать:

- Найти и исправить ошибки в коде
- Правильно обработать ошибки от API
- Улучшить качество и читаемость кода

## На что стоит обратить внимание:

- Что может вызывать проблемы с производительностью
- Работа с React Query и обработка ошибок
- Работа с состоянием
- Типизация

## Чего делать не нужно:

- Полностью рефакторить код
- Тратить время на стилизацию

## Как отправить свое задание на проверку:

- Форкнуть репозиторий на GitHub
- Внести изменения в master ветке
- Описать в README какие баги и недочеты вы обнаружили, исправили и почему
- Отправить ссылку на репозиторий в GitHub
